# 📋 Web — Доска объявлений на Flask

Полноценное веб-приложение — доска объявлений с авторизацией, личным кабинетом, категориями, лайками и интеграцией с Яндекс Картами.

---

## Стек технологий

| Компонент | Технология |
|-----------|-----------|
| Backend | Python 3, Flask |
| Авторизация | Flask-Login |
| ORM / БД | SQLAlchemy, SQLite |
| Геолокация | Яндекс Static Maps API |
| Frontend | HTML, CSS, Jinja2 |
| Деплой | Procfile (Heroku / Railway) |

---

## Функциональность

- Регистрация и авторизация пользователей (Flask-Login)
- Личный кабинет с редактированием профиля и фото
- Создание, редактирование и удаление объявлений
- Категории (Электроприборы, Авто, Недвижимость, Работа, Другое)
- Фильтрация объявлений по категории
- Система лайков («Понравившиеся»)
- Страница объявления с картой местоположения (Яндекс Maps API)
- Валидация номера телефона при регистрации
- Загрузка фотографий для профиля и объявлений
- Права доступа: редактировать/удалять может только автор или администратор

---

## Запуск

```bash
# 1. Клонировать репозиторий
git clone https://github.com/Ficus1/web.git
cd web

# 2. Создать виртуальное окружение
python -m venv venv
source venv/bin/activate      # Linux/Mac
venv\Scripts\activate         # Windows

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запустить
python main.py
```

Приложение: **http://localhost:5000**
